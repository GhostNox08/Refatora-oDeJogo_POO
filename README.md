Projeto Refatoração de Jogo POO

O StartupGame é um simulador em Java onde o usuário gerencia uma startup, tomando decisões estratégicas a cada rodada para maximizar indicadores como caixa, receita, moral e reputação. O projeto segue boas práticas de programação orientada a objetos, inclui persistência em banco H2, exportação automática de resultados em CSV, e oferece opção de jogadas automáticas via Bot.
